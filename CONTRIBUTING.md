# Contributing to NHS Clinical Safety Skills

Thank you for wanting to improve this educational resource for the UK nursing community.

## Who We Welcome

- Registered nurses and student nurses
- Digital Clinical Safety Officers (DCSOs)
- Nurse educators and practice supervisors
- Health IT developers working with NHS systems
- Clinical informaticists and FHIR specialists

## How to Contribute

### Reporting Issues

If you find an error, outdated clinical information, or a gap in the content:

1. [Open an issue](../../issues/new) and describe the problem
2. Include the skill name and the specific section
3. If possible, cite the authoritative source (BNF, NICE, NMC, NHS England)

### Improving a Skill

1. Fork the repository
2. Edit the relevant `skills/[skill-name]/SKILL.md`
3. Open a pull request with a clear description of what you changed and why
4. Include a reference to the authoritative source for any clinical content changes

### Adding a New Skill

New skills should:
- Address a genuine need for student nurses or DCSOs
- Be grounded in an authoritative NHS England, NMC, NICE, or MHRA source
- Include the educational disclaimer note at the bottom
- Follow the existing SKILL.md format (name, description, triggers, content)
- Not reproduce copyrighted material verbatim

### Clinical Content Standards

All clinical content must:
- Reference authoritative UK sources (BNF, NICE, NMC, NHS England, MHRA, Royal Colleges)
- Use UK spelling and terminology (paracetamol, adrenaline, NHS Number)
- Be appropriate for the UK health and social care system
- Include any relevant caveats (paediatric vs adult, renal impairment, pregnancy, etc.)

### What We Don't Accept

- Reproduction of DCB Information Standards text (copyright NHS England — cannot be adapted without consent)
- NHS logos, NHS England branding, or NMC logos
- Content that implies NHS England, NMC, or NICE endorsement
- Unverified clinical claims without authoritative sources
- Content that could be mistaken for official guidance

## Code of Conduct

This project follows a simple principle: **be kind and be accurate**.

We are building resources for nurses — people who care for patients. Accuracy
matters. If something is wrong, say so clearly and provide the evidence.

## Attribution

If your contribution references NHS England content, include the OGL attribution
in your pull request description:

> "Contains information from NHS England, licenced under the current version of
> the Open Government Licence."

## Questions?

Open a discussion or issue. We're a small community — someone will respond.
